#### 1. windows下用tree命令生成文件目录
```
tree /F > D:\tree.txt
```


* * *
#### 2. mac下用tree命令生成文件目录
##### 2.1. mac下用执行下面命令查看是否已经安装了tree命令
```
tree --version
```
![77b0b9b24c5a83f584dce7a8c258f746.png](evernotecid://ECD3192A-98F1-49C8-A7B1-95B1EF91F783/appyinxiangcom/22428507/ENResource/p126)
出现如图所示结果说明已经安装
如果没安装则执行下面命令安装

```
brew install tree
```
##### 2.2执行下面命令
```
tree -L 2 >README.md
```
-L 参数指定遍历层级
README.md指定输入文件
